## Manictime Integration

Модуль интеграции ERPNext с Manictime который  timesheets and assign tasks from projects

# Установка 

- Установленный frappe framwork
- Склонировать репозиторий <br/>
  `bench get-app https://github.com/K0ch3rga/erpnext-manictime-erpapp.git`
- Установить модуль в приложение <br/>
  `bench install-app manictime_integration`
- Провести миграцию <br/>
  `bench migrate`
